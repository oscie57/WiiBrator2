# WiiBrator2

 A tool that makes the Wii remote endlessly vibrate

## Building

Using the provided Makefile, just run `make`. This is assuming you have devKitPPC installed.

## Controls

| Wii Button | GC Button | Usage            |
| :--------- | :-------- | :--------------- |
| `A`        | `A`       | Toggle rumble    |
| `B`        | `B`       | Toggle pulse     |
| `Home`     | `Start`   | Exit application |
